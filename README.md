# ERP-solution

A flexible and scalable ERP system built with Laravel 11, Vue 3 (via Inertia.js), Tailwind CSS, and PostgreSQL. This project is designed to support growing businesses by streamlining operations across key departments.

# Features

**_ Modular Design: Separate modules for sales, purchases, suppliers, and shops, allowing easy extension. _**

# Shop Management:

`Create`, `edit`, and `list-shops` with details like `shop name`, `owner name`, `address`, `phone number`, `email`, `website`, `national ID`, `trade license`, `tax ID`, `notes`, and `active/inactive status`.
